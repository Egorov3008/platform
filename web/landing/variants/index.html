📁 Как применить

1. Откройте демо-страницы по ссылкам выше.

2. Если выбрали вариант — замените файл web/landing/style.css содержимым скачанного theme-*.css.

3. Базовая HTML-разметка не меняется, поэтому замена CSS безопасна для app.js.

4. Демо-файлы (demo-*.html) можно удалить после выбора.
