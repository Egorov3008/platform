## 🎨 Три Варианта

### 1. Modern Swiss 🇨🇭
**Цвет:** Глубокий тальбот (#0D7377) + турквoise (#14B8A6)
**Шрифт:** Helvetica Neue
**Настроение:** Профессиональный, утонченный, минималистичный
**Для:** Enterprise/Business пользователей
**Файлы:** `theme-swiss.css`, `demo-swiss.html`

**Характеристики:**
- Чистые линии, минимальные тени
- Тонкие границы, генерозное пространство
- Подняться при наведении на карты
- Микро-анимации, refined transitions

### 2. Dark Neon 🚀
**Цвет:** Неон розовый (#FF006E) + неон cyan (#00D9FF) на темном
**Шрифт:** Monospace (Courier Prime, IBM Plex Mono)
**Настроение:** Смелый, футуристичный, высокая энергия
**Для:** Tech enthusiasts, gaming
**Файлы:** `theme-cyberpunk.css`, `demo-cyberpunk.html`

**Характеристики:**
- Яркие неоновые свечения
- Темный фон с градиентными оверлеями
- Shimmer-анимации, intense shadows
- Uppercase текст, монопространство
- Максимальный контраст и энергия

### 3. Warm & Organic 🌿
**Цвет:** Теплый оранжевый (#D97706) + мягкий фиолетовый (#8B5CF6)
**Шрифт:** Georgia / Trebuchet MS
**Настроение:** Дружелюбный, приступный, теплый
**Для:** General consumers
**Файлы:** `theme-organic.css`, `demo-organic.html`

**Характеристики:**
- Мягкие тени, градиентные оверлеи
- Игривые анимации (spring easing)
- Rounded corners, теплые цвета
- Shine-эффекты, подняться на hover
- Accessible и humanistic

## 🧪 Тестирование

### Способ 1: Просмотр Демо
Просто откройте HTML-файлы в браузере:
```bash
# Локально
open variants/demo-swiss.html
open variants/demo-cyberpunk.html
open variants/demo-organic.html

# Или через локальный сервер
python3 -m http.server 8000
# Затем http://localhost:8000/web/frontend/variants/
```

### Способ 2: Применить на Реальный Сайт
В `index.html` замените строку:
```html
<!-- Текущий -->
<link rel="stylesheet" href="/style.css">

<!-- На один из вариантов -->
<link rel="stylesheet" href="/variants/theme-swiss.css">
<link rel="stylesheet" href="/variants/theme-cyberpunk.css">
<link rel="stylesheet" href="/variants/theme-organic.css">
```

Затем протестируйте все страницы:
1. Auth/Login
2. Dashboard (ключи)
3. Tariffs (тарифы)
4. Payments (платежи)
5. Admin (если есть)

### Способ 3: Полное Тестирование
Следуйте пошаговому гайду в `TESTING_GUIDE.md`:
- Визуальный чеклист для каждого элемента
- Проверка цветовых контрастов
- Тестирование на мобильных
- Проверка анимаций и производительности

## 🛠 Техническое Обслуживание

### Структура CSS
Все варианты используют CSS Custom Properties (переменные):
```css
:root {
    --primary: #0D7377;
    --accent: #14B8A6;
    --bg: #FFFFFF;
    --surface: #FAFBFC;
    --text: #0F172A;
    --transition: 0.25s cubic-bezier(...);
    /* и т.д. */
}
```

### Обновление Значений
Чтобы изменить цвет по всему сайту:
```css
:root {
    --primary: #NEW_COLOR; /* Обновит все первичные элементы */
}
```

## ❓ Частые Вопросы

**Q: Как применить выбранный вариант на всем сайте?**
A: Замените ссылку на CSS в `index.html` и в backend приложении (если сервирует стили).

**Q: Можно ли комбинировать элементы из разных вариантов?**
A: Да, можете взять цвета/компоненты из одного и анимации из другого. Но рекомендуется придерживаться одного.

**Q: Что если вариант не работает в IE11?**
A: CSS Custom Properties не поддерживаются в IE11. Используйте современный браузер.

**Q: Как добавить темный режим?**
A: Добавьте `@media (prefers-color-scheme: dark)` блок в CSS.

**Q: Могу ли я отключить анимации?**
A: Да, измените `--transition` на `0s` или добавьте `@media (prefers-reduced-motion)`.
