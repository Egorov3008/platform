API Роуты

REST API — FastAPI, JWT авторизация, prefix /api/v1

🔐 Auth — /api/v1/auth
Метод Путь Описание
POST /register Регистрация email + пароль
POST /login Вход email + пароль → JWT
POST /telegram/request Запрос magic link — бот шлёт ссылку в TG
POST /telegram/verify Верификация токена из ссылки → JWT
POST /refresh Обновление access token
🔑 Ключи — /api/v1/keys
Метод Путь Описание
GET / Список ключей текущего пользователя
POST / Создать ключ (тариф → 3x-UI → БД)
GET /{key_id} Детали ключа + статистика трафика
POST /{key_id}/renew Продлить ключ (новый тариф + оплата)
DELETE /{key_id} Удалить ключ из 3x-UI и БД
📦 Тарифы — /api/v1/tariffs
GET / Список доступных тарифов (публичный)
GET /{tariff_id} Детали тарифа
💳 Платежи — /api/v1/payments
POST /create Создать платёж YooKassa → ссылка оплаты
POST /webhook Webhook от YooKassa → создание ключа
👑 Админ — /api/v1/admin (только is_admin)
GET /stats Общая статистика: юзеры, ключи, выручка
GET /users Список пользователей с фильтрами
GET /users/{tg_id} Профиль пользователя + его ключи
PATCH /users/{tg_id} Блокировка / разблокировка / смена тарифа
GET /keys Все ключи системы с фильтрами
POST /keys Создать ключ вручную для пользователя
DELETE /keys/{key_id} Принудительное удаление ключа
